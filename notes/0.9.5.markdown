## Summary of changes from 0.9.4

### API

- `ReifiedRelationshipRestriction` in 0.9.5 replaces `PartialReifiedRelationship` in 0.9.4

### Concrete Syntax

In 0.9.5:

```
reifiedRelationshipRestriction <name> {
   source <domain>
   target <range>
}
```

In 0.9.4:

```
partial reifiedRelationship <name> {
   source <domain>
   target <range>
}
```
